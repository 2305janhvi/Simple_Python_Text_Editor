# Simple_Python_Text_Editor
Simple Python Text Editor
A lightweight, functional graphical text editor built using Python and the Tkinter library. This project demonstrates the fundamentals of GUI development, file handling, and event-driven programming.

🚀 Features
Create New Files: Quickly clear the workspace to start fresh.

Open Existing Files: Browse your system and load .txt files directly into the editor.

Save Functionality: Save your notes or code changes to a local file.

Clean Interface: A distraction-free UI with a large, readable font and automatic word wrapping.
